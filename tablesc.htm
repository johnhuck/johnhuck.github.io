Table 6: Observed Metadata Elements: Aggregate Level (Complete List)

  • This table is summarized by Table 5.
  • Only elements that appeared in 75% of the sample records from a given collection were included in this list.
  • Elements were organized into groupings through an intuitive affinity process.
  • The groupings were given names and organized as sub-categories of seven metadata categories identified by Lagoze, Lynch, & Daniel (1996).
  • A list of collection name acronyms is found here.
  • The right hand column gives the number of sample records from the collection that included the element.
  • Where translations and elaborations have been added, they are shown in brackets.
